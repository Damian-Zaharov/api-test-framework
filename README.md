# API Test Automation Framework (DummyJSON)

Профессиональный фреймворк для автоматизации тестирования API сервиса [DummyJSON](https://dummyjson.com).

##  Стек технологий
- **Python 3.11+**
- **Pytest** (тестовый движок)
- **HTTPX** (современный HTTP-клиент)
- **Pydantic V2** (валидация данных и схем)
- **Logging** (логирование запросов)
- **Dotenv** (управление конфигурацией)

##  Архитектура проекта
Проект построен по принципу многослойности (Layered Architecture):
- `core/`: Базовый HTTP-клиент и логгер.
- `clients/`: Низкоуровневое описание эндпоинтов.
- `services/`: Бизнес-логика и сборка сценариев.
- `schemas/`: Pydantic-модели для валидации ответов.
- `tests/`: Тестовые сценарии (Positive/Negative/CRUD).

## Установка и запуск
1. Клонировать репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустить тесты:
   ```bash
   pytest -v -s
   ```

##  Покрытие тестами (10 сценариев)
- **Auth**: Успешный логин, негативный сценарий, получение данных профиля.
- **Products**: Получение списка, поиск (позитивный/негативный), полный CRUD цикл (создание, чтение, обновление, удаление).
